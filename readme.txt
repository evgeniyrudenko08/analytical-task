Four players [P1, P2, P3 and P4] need to occupy any four cells in a 4x4 grid such that there is only one player in any cells horizontal row, any cells in vertical column, and any diagonal cells with respect to each other.
e.g. If player P1 is in cell C11, then there cannot be any player in first row and first column as well as in cells C22, C33 and C44
Grid cells are labeled as
C11, C12, C13, C14
C21, C22, C23, C24
C31, C32, C33, C34
C41, C42, C43, C44
What are the possible combinations of player position on this grid cells to satisfy above conditions.


To run please install:
 - Node 10 or higher 
 
To get all combinations:
Run in terminal, gitBash or console: 
- npm install
- npm run get-results
